# AcademicProjectPlanner
Software project planning and architectural design are critical early-stage activities in the software 
development life cycle, yet they are often performed manually, leading to inconsistencies, design flaws, 
and inefficient resource utilization especially in academic and small scale projects. Recent research 
highlights the growing role of automation, decision support systems, and AI-assisted techniques in 
improving project planning and architectural decision making. Studies on automated planning, 
architecture recommendation systems, and generative AI demonstrate that rule-based and intelligent 
tools can effectively support developers by reducing cognitive load and standardizing design outcomes. 
Building on these foundations, this work proposes an automated Academic Project Planning and High 
Level Architecture Design Tool that generates phase-wise project plans and recommends suitable 
architectural styles based on basic project parameters such as project type, team size, duration, and 
technology. Unlike complex AI-driven systems, the proposed tool adopts a lightweight, rule-based 
approach, making it easy to develop, understand, and deploy in educational environments. The tool 
aligns with existing research on architectural decision support while focusing on simplicity, 
transparency, and academic usability. This approach demonstrates that even minimal automation can 
significantly enhance planning accuracy and architectural clarity for novice developers and students, 
while serving as a foundation for future intelligent extensions.
